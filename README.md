# 💬 Real-Time Chat Platform

A high-performance, real-time communication platform allowing users to seamlessly create, join, and message inside dynamic channels. Built with sub-millisecond data synchronization between clients.

## 🚀 Key Features

* **Instant Messaging:** Real-time message broadcasting using WebSockets.
* **Dynamic Rooms:** Users can easily join or switch between chat rooms using unique room IDs.
* **Responsive UI:** Clean, modern interface designed for both mobile and desktop screens.
* **Persistent Connection:** Auto-reconnection logic for low-latency network performance.

## 🛠️ Tech Stack

* **Frontend:** React, HTML5, CSS3 / Tailwind CSS
* **Backend:** Node.js, Express
* **Real-Time Engine:** Socket.io (WebSockets)

## 📦 Installation & Setup

Follow these steps to run the application locally:

1. Clone the repository:
   ```bash
   git clone https://github.com
   ```

2. Install dependencies for both the frontend and backend:
   ```bash
   npm install
   ```

3. Start the local development server:
   ```bash
   npm run start
   ```
